Project Title     :   Precision rescue intelligence monitoring system (PRIAMS)


Project overview  :  

            The Precision Rescue Ambulance Intelligence Monitoring System is designed to expedite the movement of ambulances through congested traffic. By leveraging GPS technology, the system identifies the location of approaching ambulances and communicates this information to nearby traffic signals. When an ambulance is within a specified distance (e.g., 2 kilometers) of a traffic light, the signal is automatically turned red to clear the intersection, allowing the ambulance to proceed without delay.

My role      :

            Code Development:Developed the primary code for GPS tracking using the TinyGPS++ library to accurately monitor ambulance locations. Established a reliable communication link between ambulances and traffic signal control systems to prioritize emergency vehicles.

            Designed and Executed Wiring Setup :Created and implemented the wiring configuration for the Arduino Uno board, ensuring seamless integration of all hardware components for optimal system performance.

            Debugging :Debugged TinyGPS++ and Arduino code to enhance the functionality of IoT devices used in PRIAMS



Project Title     :   Dna Based Encryption System


Project overview  :  

            This project implements a novel encryption system that leverages DNA sequences for enhanced data security. Unlike traditional methods that rely on fixed DNA sequences, this system generates a unique 128-bit DNA sequence for each encryption cycle. The dynamic generation of DNA sequences makes the system more secure and harder to break, addressing the limitations of earlier encryption systems that used fixed sequences.

My role      :

            Code Development: developed a DNA-based encryption system using Python, incorporating advanced cryptographic techniques. Additionally, I implemented socket programming to facilitate more convenient data transmission within this encrypted system.
